# Lista de Plugins de Neovim

## Gestión de Plugins
- **lazy.nvim** - Gestor de plugins (lazy loading)

## Temas de Colores
- **gruvbox.nvim** - Tema Gruvbox
- **visual_studio_code** - Tema Visual Studio Code
- **darcula-dark.nvim** - Tema Darcula Dark
- **oxocarbon.nvim** - Tema Oxocarbon
- **tender.vim** - Tema Tender

## Autocompletado y LSP
- **nvim-cmp** - Motor de autocompletado
- **nvim-lspconfig** - Configuración del Language Server Protocol
- **none-ls.nvim** - Formateadores y linters

## Árbol de Sintaxis
- **nvim-treesitter** - Parsing y resaltado de sintaxis mejorado
- **nvim-treesitter-context** - Muestra contexto (clases/funciones) arriba

## Búsqueda y Navegación
- **fzf-lua** - Búsqueda fuzzy de archivos y texto
- **flash.nvim** - Saltos rápidos con resaltado visual
- **which-key.nvim** - Muestra atajos de teclado disponibles

## Interfaz y UI
- **lualine.nvim** - Barra de estado personalizada
- **bufferline.nvim** - Barra de pestañas para buffers
- **neo-tree.nvim** - Explorador de archivos tipo árbol
- **dropbar.nvim** - Breadcrumbs de navegación
- **nvim-scrollbar** - Barra de desplazamiento con indicadores
- **noice.nvim** - Mejora mensajes y notificaciones del LSP
- **indentLine** - Muestra líneas de indentación
- **mini.indentscope** - Guías de indentación animadas

## Git
- **vim-fugitive** - Integración con Git
- **vim-signify** - Muestra cambios de Git en la línea de signos

## Testing
- **neotest** - Framework de testing (Python, etc.)

## Utilidades de Código
- **nvim-autopairs** - Auto-cierra paréntesis, llaves, comillas
- **nvim-ts-autotag** - Auto-cierra y renombra tags HTML/XML
- **vim-illuminate** - Resalta todas las ocurrencias de la palabra bajo el cursor
- **vim-prettier** - Formateador de código Prettier
- **eslint.nvim** - Integración con ESLint
- **go.nvim** - Soporte mejorado para Go

## Portapapeles
- **nvim-osc52** - Copia al portapapeles del sistema remoto (SSH)
